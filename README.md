# Remove Element App

This is a simple web application that allows you to remove a specific element from an array of numbers. The app is built using HTML, CSS, and JavaScript.

## How to Use

1. Clone this repository to your local machine or download the source code as a ZIP file.

2. Open the `remove_element_app.html` file in a web browser.

3. Enter a comma-separated list of numbers into the "Enter comma-separated numbers" input field. For example: `3, 2, 2, 3`.

4. Enter the value that you want to remove from the array into the "Value to remove" input field.

5. Click the "Remove" button.

6. The app will display the modified array without the specified value and its new length.

## Compatibility

The app is compatible with modern web browsers such as Chrome, Firefox, Safari, and Edge.
